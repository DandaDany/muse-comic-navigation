# 漫博全方位導航客服（離線版）

手機優先、可直接離線開啟的 2026 漫畫博覽會展場導航工具。

## 功能

- 搜尋 IP 後選擇販售廠商；展示或販售尚未確認的資料會明確標示
- 直接搜尋品牌、攤位、舞台、入口附近設施等目的地
- 從入口定位，或用左右兩側品牌判斷目前位置與面向
- 依起點與目的地重新計算最短走道，並顯示文字路線與地圖紅線
- 收錄 63 個不重複廠商群組、378 筆明確販售 IP、66 筆展示／待確認資料
- 不需要 API、伺服器或網路連線

## 使用方式

直接開啟根目錄的 `index.html` 即可。它已內嵌完整介面、程式、資料與地圖，即使從 ZIP 內直接點開，也不需要另外載入旁邊的檔案。

`app.css`、`app.js`、`data.js` 與 `assets` 保留作為原始碼，方便後續維護；一般使用者不需要開啟它們。

## 資料範圍

- 活動：2026 第25屆漫畫博覽會
- 日期：2026/7/23–7/27
- 地圖：主辦單位 2026 官方展場平面圖
- 品牌與 IP：漫畫博覽會官網新聞整理資料

正式使用時，若現場有排隊護欄、臨時封路或工作人員引導，請以現場動線為準。標示為「位置待確認」的廠商仍可搜尋資料，但不會產生可能誤導的路線。
