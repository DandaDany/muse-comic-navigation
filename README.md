# 木棉花導航小幫手

手機優先、可離線使用的展場導航工具。使用者選擇左手邊與右手邊看到的品牌後，系統會依照使用者當下朝向提供前往木棉花 MUSE 的文字路線，並在平面圖上標示目前位置紅點與推薦路線紅線。

## 目前功能

- 手機版 UI，左手品牌在左、右手品牌在右
- 選定其中一側品牌後，另一側只顯示相鄰且可定位的品牌
- 依左右品牌順序判斷使用者面朝方向
- 以使用者視角回答直走、左轉、右轉或向後轉
- 地圖顯示目前位置紅點、面向箭頭及推薦路線紅線
- 品牌別名搜尋與入口導航模式
- 地圖縮放、拖曳與全螢幕檢視
- 不需要 API 或後端

## 使用方式

直接開啟 `index.html`。下載整個 repository ZIP 並解壓縮後，也可以在沒有網路的環境開啟；請保留 `index.html`、`assets`、`js` 的相對位置。

## 專案結構

```text
muse-comic-navigation/
├── index.html
├── README.md
├── assets/
│   └── map-part-*.js
└── js/
    ├── app-part-*.js
    └── app-run.js
```

為了能在 GitHub 上維護，同時保留離線使用，平面圖與程式被拆成多個小型 JavaScript 資料片段，瀏覽器開啟時會自動組合。
